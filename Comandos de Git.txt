PARA INICIAR UN REPOSIORIO:

Primer caso.
Cuando trabajo de forma local
git ini
Segundo caso.
Cuando trabajo con un repositorio web
git clone repositorio

PASOS A SEGUIR PARA CADA COMMIT:

1. git status
2. git add <archivo> o . para todo
3. git commit -m "mensaje" 

PASOS PARA SUBIR A LA WEB:

1. git push origin master
